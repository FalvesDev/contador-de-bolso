# Roadmap - Contador de Bolso

## Status: MVP Funcional
Data: Março 2026

---

## 🔴 PRIORIDADE ALTA (Essencial)

### 1. [ ] Persistência Local (AsyncStorage)
- Salvar transações no dispositivo
- Dados não se perdem ao fechar o app
- Sincronizar com Supabase quando online

### 2. [ ] Editar/Excluir Transações
- Modal de edição ao clicar na transação
- Botão de excluir com confirmação
- Atualizar lista em tempo real

### 3. [ ] Gráficos Visuais
- Gráfico de pizza (gastos por categoria)
- Gráfico de barras (comparativo mensal)
- Gráfico de linha (evolução do saldo)

### 4. [ ] Filtros na Lista de Transações
- Por período (hoje, semana, mês, personalizado)
- Por categoria
- Por tipo (receita/despesa)
- Por valor (maior/menor que)

### 5. [ ] Busca de Transações
- Campo de pesquisa na tela de transações
- Buscar por descrição
- Resultados em tempo real

---

## 🟡 PRIORIDADE MÉDIA (Importante)

### 6. [ ] Onboarding
- Telas de boas-vindas (3-4 slides)
- Explicar funcionalidades principais
- Configuração inicial (nome, renda mensal)

### 7. [ ] Exportar Dados
- Gerar CSV real
- Gerar PDF com relatório
- Compartilhar via WhatsApp/Email

### 8. [ ] Animações Suaves
- Transições entre telas
- Animação ao adicionar transação
- Skeleton loading

### 9. [ ] Haptic Feedback
- Vibração ao salvar transação
- Feedback tátil nos botões
- Confirmar ações importantes

### 10. [ ] Pull to Refresh
- Puxar para atualizar dados
- Indicador de carregamento
- Sincronizar com backend

---

## 🟢 PRIORIDADE BAIXA (Diferencial)

### 11. [ ] Entrada por Voz Funcional
- Integrar Speech-to-Text
- Processar texto com IA
- Extrair valor, categoria, descrição

### 12. [ ] Widgets
- Widget de saldo na home
- Widget de gastos do dia
- Atualização automática

### 13. [ ] Notificações Push
- Lembrete de contas a pagar
- Alerta de orçamento
- Resumo semanal

### 14. [ ] OCR de Notas Fiscais
- Fotografar recibo
- Extrair dados automaticamente
- Sugerir categoria

### 15. [ ] Open Finance
- Conexão com bancos
- Importar transações automático
- Atualização de saldo em tempo real
